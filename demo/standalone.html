🔀 WebTAK Channels — Standalone Test Harness

Exercises the full Channels panel against a MOCKED Marti groups API (no TAK server needed).

1 · Mock the Marti API

This page stubs window.fetch for /Marti/api/groups/all and /Marti/api/groups/active with an in-memory group list, so you can exercise toggling, filtering, and the optimistic-update/revert-on-failure path without a live server.

2 · Open the panel

Notes

• Against a real TAK server, drop the mock and the plugin talks to GET /Marti/api/groups/all / PUT /Marti/api/groups/active on the same origin, using the operator's existing session cookie.
• Drag the panel by its header; use the search box to filter by channel name.
• Flipping a switch applies immediately (optimistic update), same as ATAK's Channels tool.
